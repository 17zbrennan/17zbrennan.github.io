Bearhand Bill was a game created in actionscript. This game gave an isometric view of the character, Bill, who had bear hands. The main gameplay features of this game was the movement, the shooting, and the reloading. Due to this being built in actionscript 3.0, most of the gameplay mechanics and features had to be programmed by hand.
The Java Maze Builder was created using Java. This program would create randomly generated mazes via an array. It would then place a start and end point for it to find. It accomplished this in two different ways. It would either use a stack, which would be an extremely inefficient way to do so, or it would use a queue. When it used the queue it would find the best path possible.
