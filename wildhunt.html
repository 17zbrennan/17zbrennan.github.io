Wild Hunt

This is a 2D-Platformer created with Unity. The main draw for this game would be the modular procedural generation. This is accomplished by pulling from a group of game objects and placing them in the correct position. Each time this game is played, there will be a different map to explore. This game was completely created by Zachary Brennan.
